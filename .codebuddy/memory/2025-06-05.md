## 修复：创建订单新顾客联系方式验证

**问题**：创建订单时，如果顾客是新顾客且没有填写联系方式，点击创建会报错。

**修复内容**（src/pages/OrdersPage.tsx）：
- 添加了对新顾客联系方式的验证
- 检查是否有未填写完整的账号（只选了平台没填账号ID，或相反）
- 如果联系方式不完整，给出明确提示而不是直接报错
- 过滤有效账号时要求 platform 和 accountId 都必须存在

**代码变更**：
```typescript
// 验证联系方式：检查是否有未填写完整的账号
const hasInvalidAccount = newCustomerAccounts.some(a => 
  (a.platform && !a.accountId.trim()) || (!a.platform && a.accountId.trim())
)
if (hasInvalidAccount) {
  alert('请填写完整的联系方式（平台和账号ID）')
  return
}

// 过滤掉完全空的账号
const validAccounts = newCustomerAccounts.filter(a => a.platform && a.accountId.trim())
```

## 修复：预约时间默认值

**问题**：预约时间默认从18:00开始，应该全天可选且默认是当前时间的整点。

**修复内容**（src/pages/OrdersPage.tsx）：
- 修改 handleOpenDialog 函数，设置默认日期为今天
- 设置默认时间为当前小时的整点（如当前11:45则默认11:00）

**代码变更**：
```typescript
const handleOpenDialog = () => {
  // 获取当前时间并设置为整点
  const now = new Date()
  const currentHour = String(now.getHours()).padStart(2, '0')
  const currentDate = now.toISOString().split('T')[0]
  
  setFormData({
    ...
    appointmentDate: currentDate,
    appointmentTime: `${currentHour}:00`,
    ...
  })
}
```
